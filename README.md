# UntactCycling
### プロジェクト概要

本プロジェクトは、コロナ禍において非対面での運動を支援することを目的に、
卒業制作として開発したシステムです。
自転車走行時の速度および心拍数を取得し、Unity上で可視化する仕組みを構築しました。

### 開発背景

指導教員の助言をきっかけに、
「精神的健康の維持には日常的な活動量の確保が重要である」という考えのもと、
感染対策を行いながら一人でも運動できる仕組みとして本システムを制作しました。

### 担当内容

Arduinoを用いた速度・心拍数の計測プログラムの実装

Raspberry PiからFirebaseへ計測データを送信する処理の実装

Firebaseを簡易的なバックエンドとして設計し、
速度・心拍数の状態データを管理

UnityからFirebaseのデータを取得し、
オブジェクトのプロパティと連動させた可視化処理を実装

### システム構成

センサー（速度・心拍）

Arduino（計測処理）

Raspberry Pi（データ送信）

Firebase（データ管理）

Unity（可視化）

### 工夫した点・学んだこと

ハードウェアとソフトウェアを連携させるシステム構成を設計・実装

デバイス間通信やAPIを意識したデータ連携を経験

Unityを用いたデータ連動型の可視化処理を実装し、
システム全体の流れを意識した開発を行いました
